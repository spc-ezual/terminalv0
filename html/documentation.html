Documention

Information générale:


-Ce programme est destinée à un usage privée uniquement tout droits reservés
-Une connexion internet stable est nessecaire au bon fonctionnement du programme

Comment utiliser:


1)Lancer le programme
2)A l'aide de votre clavier, taper les mots clef en suivant l'arborescence noter dans le fenêtre d'entrée.
3)-Dans le cas d'une recherche a partir d'un artiste, noter le nom de l'artiste et attendre le résultat.
-Dans le cas d'une recherche a partir d'un titre, noter le nom du titre et attendre le résultat.
4)Pour arreter le probramme, taper stop à la première étape de l'arborescences.

listes des méthodes :


-info_artiste(artiste_objet):
  Méthode qui a pour but d'aller chercher dans l'api musicbrainz les info principales sur l'artiste et de les renvoyer
  Renvoie les infos de l'artiste
-musique_artiste(artiste_objet):
  Méthode qui a pour but d'aller chercher dans l'api musicbrainz les musique créer/écrite/participer l'artite et de les renvoyer
  Renvoie les musique créer par l'artiste
-collab(artiste_objet):
  Méthode qui a pour but d'aller chercher dans l'api musicbrainz toute les collaboration que l'artiste a fais avec d'autres artistes et de les renvoyer
  Renvoie les collaboration de l'artiste
-artiste musique(musique_objet):
  Méthode qui a pour but d'aller chercher dans l'api musicbrainz le ou les artistes qui ont creer cette musique et de les renvoyer
  Renvoie le(s) qui a(ont) travaillé sur la musique
-info_musique(musique_objet):
  Méthode qui a pour but d'aller chercher dans l'api musicbrainz les info principale de la musique et de les renvoyer
  Renvoie les infos de la musiques
